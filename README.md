# ぱっとみえ! — フォトリーディング・トレーニングゲーム

3〜6歳の子供向けの、カードの配置・順番を記憶して再現するiPad用ゲームアプリ(PWA)です。
仕様は [PRD.md](PRD.md) を参照してください。アプリ本体は `app/` フォルダにあります。

## 遊び方

- **ぱっとみて あそぶ**: グリッドに並んだカードを記憶し、シャッフルされたカードをドラッグして元の位置に戻す
- **じゅんばん あそび**: カードが出てきた順番を記憶し、その順にタップする
- **ごほうび ずかん**: 遊んで貯めた⭐で、コレクションのイラストが順番に解放される
- **せってい**(右下の⚙️): ペアレンタルゲート付き。わくのかず・おぼえるじかん・えがら・おまかせモードなどを変更できる

## iPadで使う方法

1. このMacで `app/` フォルダをローカルサーバーで公開する:
   ```sh
   cd "app" && python3 -m http.server 8123
   ```
2. iPadのSafariで `http://<MacのIPアドレス>:8123` を開く(同じWi-Fiに接続していること)
3. 共有ボタン → **「ホーム画面に追加」** をタップ
4. ホーム画面のアイコンから起動すると、フルスクリーンのアプリとして動作します

> 常時サーバーを立てたくない場合は、`app/` フォルダをGitHub PagesやNetlifyなどの無料静的ホスティングに置いても同じ手順で使えます。一度ホーム画面に追加すればService Workerによりオフラインでも動作します。

## 技術構成

- HTML / CSS / JavaScript のみ(フレームワーク・ビルド不要)
- カード絵柄: 絵文字 + SVGピクトグラム(外部素材なし)
- 効果音: Web Audio APIで合成 / 音声ガイド: Web Speech API(日本語)
- データ保存: localStorage(完全ローカル、外部通信なし)
